# Automação de Relatório Fiscal Mensal com Python

Este projeto simula uma automação fiscal para consolidar notas fiscais de venda, calcular tributos estimados, identificar inconsistências e gerar relatórios mensais de apoio à conferência fiscal.

A ideia é representar uma rotina comum em áreas fiscais, financeiras ou de backoffice, onde arquivos mensais precisam ser consolidados e analisados antes do fechamento.

## Objetivo

Automatizar uma rotina que normalmente seria feita manualmente em planilhas.

O pipeline realiza:

- leitura de arquivos mensais de notas fiscais;
- consolidação das bases;
- padronização dos dados;
- cálculo de valores líquidos;
- cálculo estimado de ICMS, PIS e COFINS;
- identificação de inconsistências fiscais;
- geração de tabelas resumo;
- criação de gráficos;
- geração de relatório final;
- registro de log de execução.
